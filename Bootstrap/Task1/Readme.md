# 🏫 Responsive Student Management Portal

A clean, mobile-friendly multi-page web portal designed for managing student records, tracking attendance, and handling campus registrations. Built entirely using HTML5, CSS3, and Bootstrap 5.

## 📱 Project Structure

The portal consists of 5 responsive pages:
*   **Home Page (`index.html`):** Features an interactive hero carousel, a 3-card services grid, and an FAQ accordion.
*   **Registration Form (`register.html`):** A student onboarding form equipped with native Bootstrap form validation.
*   **Dashboard (`dashboard.html`):** Displays key statistics cards and a responsive data table with a dynamic search bar.
*   **Student Profile (`profile.html`):** A clean card layout showing individual student details and profile images.
*   **Contact Us (`contact.html`):** An inquiry form featuring interactive form alerts.

## 🛠️ Tech Stack & Components Used

*   **Core Stacks:** HTML5, CSS3, JavaScript (ES6)
*   **UI Framework:** Bootstrap 5 (Grid System, Flexbox utilities)
*   **Key Components:** Navbar, Carousel, Cards, Forms, Tables, Modals, Accordions, Badges, and Alerts.
