# 🎓 Online Course Registration Portal

A responsive, multi-page web application designed for a training institute. It allows students to browse technical training tracks, check course packages, review faculty credentials, and submit enrollment applications online. Built using HTML5, CSS3, and Bootstrap 5.

## 📱 Project Structure

The platform is organized into 5 fully responsive pages:
*   **Home Page (`index.html`):** Features an interactive hero banner, a 6-course featured catalog grid, student testmonial sections, and an authentication login modal.
*   **Courses Page (`courses.html`):** Displays all core curriculum offerings (Web Development, Python, Data Science, AI/ML, Java, and Cloud Computing) in a uniform Bootstrap card grid.
*   **Registration Page (`register.html`):** An onboarding application form with document attachment handling and a responsive success alert state upon submission.
*   **About Us (`about.html`):** Shares the institute's vision, history, and faculty team profiles using image card panels.
*   **Contact Us (`contact.html`):** Houses the user inquiry form, physical campus addresses, and an embedded map iframe placeholder.

## 🛠️ Tech Stack & Components Used

*   **Core Stacks:** HTML5, CSS3, JavaScript (ES6)
*   **UI Framework:** Bootstrap 5 (Grid Utility, Responsive Flexbox)
*   **Key Components:** Navbar, Carousel, Cards, Forms (Input Groups), Modals, Buttons, Grid Columns, and Interactive Alerts.
