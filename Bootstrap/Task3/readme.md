# 🏫 College Event Management System

A responsive, multi-page web application designed to streamline campus activity coordination. It allows students to browse upcoming fests, check timing schedules, view past event galleries, and submit automated registration applications. Built using HTML5, CSS3, and Bootstrap 5.

## 📱 Project Structure

The portal is organized into 7 clean, responsive pages:
* **Home Page (`index.html`):** Features the campus event banner carousel, live activity highlights, and call-to-action buttons.
* **Events Page (`events.html`):** Displays 6 core events (Technical Quiz, Hackathon, Poster Presentation, Coding Contest, Project Expo, and Cultural Fest) using structured card templates.
* **Registration Page (`register.html`):** A student onboarding form with dynamic dropdowns for event selection and a success message pop-up state.
* **Schedule Page (`schedule.html`):** Displays structural timing parameters using responsive, clean `table-striped` and `table-hover` layout styles.
* **Gallery Page (`gallery.html`):** Showcases high-quality campus photographs in a clean, fluid grid layout using responsive image cards.
* **FAQ Page (`faq.html`):** Utilizes the Bootstrap Accordion component to handle questions about fees, eligibility, and multi-event participation rules.
* **Contact Page (`contact.html`):** Houses the user inquiry form, student coordinator point-of-contact details, and social media anchor handles.

## 🛠️ Tech Stack & Components Used

* **Core Stacks:** HTML5, CSS3, JavaScript (ES6)
* **UI Framework:** Bootstrap 5 (Responsive Grid, Utilities)
* **Key Components:** Navbar, Carousel, Cards, Forms, Tables, Accordion, Alerts, Buttons, and Modals.
